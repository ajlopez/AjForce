# My Company

Simple index page with:

- Express
- MongoDb
- Ejs for views
- Ejs-locals for view layout
- Twitter Bootstrap for styles

## Install

```
npm install
```

## Run

```
node app
```
Locally starts as `http://localhost:3000`.
